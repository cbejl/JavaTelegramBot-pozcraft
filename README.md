# Java Telegram Bot для моего проекта ПОЦКРАФТ

---

Приниамет идеи/вакансии и отправляет их администрации с необходимой пометкой
(Пользователь тыкает на кнопочки под сообщением бота. Реализовано через редактирование последнего сообщения бота, а не отправку новых)

---
***Для работы необходимо заполнить конфиг (resources/application.properties)***

*bot.admins - long значение (!не @имя_пользователя!)*
